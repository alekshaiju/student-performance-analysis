# 📊 Student Performance Analysis

## 🔍 Overview
This project analyzes student data to understand how study habits and attendance affect performance.

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📁 Dataset
Contains student information like:
- Study hours
- Attendance
- Marks

## 📈 Features
- Data analysis using Pandas
- Visualization using graphs
- Insights on student performance

```bash
pip install pandas matplotlib seaborn
